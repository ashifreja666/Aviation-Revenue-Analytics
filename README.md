# AI-Driven Airlines Data Analytics Project ✈️

## Project Overview
This project is an end-to-end data analytics solution designed to extract actionable business intelligence from raw airline database extracts. By establishing a relational database environment in **PostgreSQL** and connecting it to **Advanced Excel** for dynamic visualization, this project uncovers critical insights regarding revenue generation, fleet utilization, and route performance. 

To maximize efficiency and ensure enterprise-grade standards, **Artificial Intelligence (LLMs)** was integrated into the workflow as a development copilot—accelerating SQL query formulation, validating data models, and structuring executive presentations.

## 🎯 Business Objective
The primary goal of this analysis is to evaluate key performance indicators (KPIs) across ticketing and flight data to help airline stakeholders:
1. Identify peak booking behaviors to optimize pricing strategies.
2. Evaluate revenue distribution across different fare classes and aircraft models.
3. Isolate top-performing and underperforming flight routes for network planning.

## 🛠️ Technology Stack
* **Database Management:** PostgreSQL (Data modeling, Joins, Aggregations)
* **Data Visualization & Processing:** Microsoft Excel (Power Query, Data Model, Pivot Tables, Interactive Dashboards)
* **Workflow Acceleration:** Generative AI (Prompt engineering for SQL optimization, syntax debugging, and presentation copywriting)

## 🤖 AI Integration Strategy
In the modern analytics landscape, efficiency is as critical as accuracy. AI was actively leveraged in this project to:
* **Accelerate Query Development:** Utilized AI to rapidly prototype complex PostgreSQL queries, specifically handling date/time extractions and nested subqueries.
* **Data Validation:** Acted as a secondary validation layer to ensure SQL outputs perfectly matched Excel Data Model calculations.
* **Business Storytelling:** Assisted in translating raw data outputs into structured, executive-facing insights and presentation materials.

## 📊 Key Insights & Deliverables
* **Dynamic Interactive Dashboard:** Built a highly interactive Excel dashboard utilizing slicers connected to multiple pivot charts, allowing users to filter metrics by booking timelines.
* **Revenue Drivers:** Identified that while Economy class drives the vast majority of ticket volume, Business class contributes a disproportionately high percentage of total revenue.
* **Route Optimization:** Highlighted the bottom 5 underperforming routes, providing data-backed evidence for potential route cancellation or marketing intervention.

## 📂 Repository Structure
* `data/`: Contains the raw CSV data files used for the database.
* `sql_queries/`: Contains the SQL scripts used to validate KPIs.
* `dashboard/`: Contains the final interactive Excel workbook.
* `presentation/`: Contains the executive summary PDF.
